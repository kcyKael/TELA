# Online Apparel Store

A PHP + MySQL e-commerce web application developed as a final project for
CCS0043 - Applications Development and Emerging Technologies.

## Tech Stack

- PHP (Procedural)
- MySQL (MySQLi)
- Bootstrap 5
- HTML
- CSS
- JavaScript

## Project Structure

- admin/ - Seller administration
- auth/ - Authentication
- buyer/ - Buyer pages
- assets/ - CSS, JavaScript, images, icons
- config/ - Configuration files
- database/ - SQL scripts
- includes/ - Shared PHP components
- uploads/ - Uploaded assets
- docs/ - Project documentation

## Setup

1. Place the project inside your web server directory.
2. Import the database SQL file.
3. Configure the database connection in config/database.php.
4. Start Apache and MySQL.
5. Access the project through your browser.
